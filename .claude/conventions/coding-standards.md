# Coding Standards

**ALL AGENTS MUST FOLLOW THESE CONVENTIONS**

## General Conventions

-   ** IMPORTANT ** Use tabs (not spaces) for indentation.
-   When writing scripts (non-bash), use javascript rather than python.
